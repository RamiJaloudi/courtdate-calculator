# CourtDate Calculator

React + TypeScript legal-tech app for calculating litigation deadlines from filing dates, service dates, and court rules.

## Highlights

- TypeScript date-rule engine
- React UI with jurisdiction selector
- Holiday/weekend adjustment logic
- Vitest unit tests

## Run locally

```powershell
npm install
npm run dev
npm test
```

## Tech stack

Vite, React, TypeScript, Vitest.
